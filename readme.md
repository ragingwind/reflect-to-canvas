# reflect-to-canvas

> Reflect your image to canvas


## Install

```
$ npm install --save reflect-to-canvas
```


## Usage

```js
document.querySelector('#srcImg').reflectToCanvas(document.querySelector('#reflect'), {
	reflection: 0.5,
	bgcolor: '#000000',
	width: 544,
	height: 184
});
```
